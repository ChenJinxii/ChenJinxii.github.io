3. Github部分

  • 注册Github账号

  • 创建Repository，名字为==username.github.io==(username是你的用户名)

  • 配置Github SSH密钥

    • 桌面空白处右键选择Git bash Here

    ​ 回车之后除了账号密码，建议都直接回车，其中有一个存放密钥的路径，打开这个路径下的id_rsa.pub，复制其中的所有内容。

    • 点击github头像下的Setting –> SSH and GPG keys –> New SSH key –> 粘贴刚刚复制的内容
